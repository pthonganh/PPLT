# FAMILY LIBRARY MANAGEMENT

## Description
This is a console application written in Python to manage a family library.

## Features
- Add new books
- Display all books in table format
- Search book by ID
- Sort books by year
- Show statistics (total books, total quantity)
- Save and load data from TXT file

## How to run
Open terminal and run:

```bash
python Chapter3_Mini_Project/app.py
